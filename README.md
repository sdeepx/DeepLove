<h2 align="center"> Love Percentage Calculator </h2>

<h3> Installation* <h3>
**NOTE**: Python 3.6 or higher is required. 

        # clone the repo
         $ git clone https://github.com/shubha161/DeepLove
         
        # change the working directory to DeepHash
         $ cd DeepLove
<h3> Usage </h3>
        
        # Run the file on terminal
        $ python3 DeepLove.py
